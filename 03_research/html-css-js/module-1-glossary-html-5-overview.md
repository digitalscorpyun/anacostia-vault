# 🧠 Module 1 Glossary: HTML5 Overview

**📘 Course**: IBM AI Developer – _Introduction to HTML, CSS, & JavaScript_

A reference glossary for core concepts introduced in Module 1. Ideal for review, linking to deeper notes, or turning into flashcards.

| Term      | Definition |
|-----------|------------|
| **DOM Tree** | *Document Object Model* — the data representation of all elements in a webpage. It structures HTML as a tree of objects that can be manipulated via scripting. |
| **XML**       | *eXtensible Markup Language* — a flexible text format designed for storing and transporting data. It allows users to define custom markup tags. |
| **XHTML**     | *eXtensible Hypertext Markup Language* — a stricter, XML-based version of HTML, ensuring cleaner formatting and more consistent browser behavior. |
