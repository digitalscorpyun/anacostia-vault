# 🌐 Module 1 Cheatsheet: HTML5 Overview

A quick-reference sheet for common HTML5 elements covered in Module 1 of the IBM AI Developer course. Use this as a handy reference or integrate into your HTML notes.

| Element | Description | Example |
|--------|-------------|---------|
| `<!-- -->` | Denotes a comment in HTML (not visible in browser). | `<!-- This is a comment -->` |
| `<!DOCTYPE html>` | Declares the document type at the top of HTML documents. | `<!DOCTYPE html>` |
| `<html>` | Root tag that wraps the entire HTML document. | `<html> ... </html>` |
| `<head>` | Contains metadata like `<title>`, `<meta>`, and linked CSS. | `<head> ... </head>` |
| `<body>` | Contains the main content of the HTML page. | `<body> ... </body>` |
| `<h1>` | Top-level heading tag. | `<h1>Main Title</h1>` |
| `<p>` | Paragraph of text. | `<p>This is a paragraph.</p>` |
| `<a href="">` | Anchor tag to create hyperlinks. | `<a href="https://www.ibm.com">IBM</a>` |
| `<img src="">` | Embeds an image. Requires `src`, `alt`, and optional size. | `<img src="logo.png" alt="Logo" width="300">` |
| `<div>` | Generic container, often used with CSS. | `<div class="section">...</div>` |
| `<ul>` | Creates an unordered (bulleted) list. | `<ul><li>Item 1</li></ul>` |
| `<ol>` | Creates an ordered (numbered) list. | `<ol><li>Step 1</li></ol>` |
| `<li>` | List item for use inside `<ul>` or `<ol>`. | `<li>Item</li>` |
| `<table>` | Starts a table block. | `<table> ... </table>` |
| `<tr>` | Table row. | `<tr> ... </tr>` |
| `<th>` | Table header cell. | `<th>Header</th>` |
| `<td>` | Table data cell. | `<td>Data</td>` |
| `<link>` | Links an external file (like a CSS stylesheet). | `<link rel="stylesheet" href="style.css">` |
| `<meta>` | Provides metadata about the document. | `<meta charset="UTF-8">` |
| `<script>` | Embeds or links to JavaScript. | `<script>alert("Hello");</script>` |
| `<title>` | Defines the page title shown in the browser tab. | `<title>My Page</title>` |
