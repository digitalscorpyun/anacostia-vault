## 🔥 **Installation: Sankofa CLI Theme**  

### **1. Clone the Rebellion**  
```bash  
# Install Sankofa theme from AfricCyber repos  
git clone https://github.com/africcyber/sankofa-terminal.git ~/.sankofa-terminal  
```  

### **2. Activate the Ancestors**  
Add to `~/.bashrc` or `~/.zshrc`:  
```bash  
# Load Sankofa theme  
source ~/.sankofa-terminal/sankofa.theme  

# Adinkra Prompt Symbol (default: 🡲)  
export SANKOFA_PROMPT="𓃒"  # Ankh symbol for life-force coding  
```  

### **3. Test the Waters**  
Restart terminal → Prompt should now display:  
```  
𓃒 /anacostia-vault $  
```
