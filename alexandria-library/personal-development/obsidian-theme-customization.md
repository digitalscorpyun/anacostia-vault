```css
/* 1.1: Root Color Scheme */
:root {
  --background-primary: #f0e6ff;
  --text-normal: #2e1a46;
  --text-accent: #4B0082; /* Royal Purple */
}

/* 1.1a: Graph Customization */
.graph-view.color-fill {
  color: #4B0082 !important;
}

/* 1.2: Typography */
body {
  font-family: Georgia, serif;
}
pre, code {
  font-family: 'Fira Code', monospace;
}

/* 1.2a: Tag Visibility */
.tag {
  background-color: #4B0082;
  color: white;
  padding: 2px 6px;
}
```
- Enable in Obsidian: Settings > Appearance > CSS snippets > Enable `custom-theme`.

#### 4. Automation Script
Create `theme_backup.py` in `anacostia-vault/projects`:
```python
import os
import shutil
import datetime

def backup_theme(directory):
    css_path = os.path.join(directory, ".obsidian", "snippets", "custom-theme.css")
    backup_dir = os.path.join(directory, "backups", "themes")
    os.makedirs(backup_dir, exist_ok=True)
    timestamp = datetime.datetime.now().strftime("%Y-%m-%d_%H-%M-%S")
    backup_path = os.path.join(backup_dir, f"custom-theme_{timestamp}.css")
    shutil.copy2(css_path, backup_path)
    print(f"Backed up to {backup_path}")

if __name__ == "__main__":
    backup_theme("anacostia-vault")
```
- Run to create encrypted backups (add encryption with `cryptography` library if needed).
