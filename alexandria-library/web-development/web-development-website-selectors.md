## 🌍 **Selector Deployment**

🔹 **🛡 Purpose – Why Selectors Rule**  
- **Targeting**: Pinpoint DOM elements—e.g., `<div id="content">`.  
- **Styling**: CSS selectors style *Lion* dashboards—[[anacostia-vault-execution]].  
- **Automation**: JS selectors scrape bias data—[[bias-flag-script]].  
- **Justice**: Ensure accessibility—audit UI for equity—[[ai-ml-ethics-case-studies]].  

🔹 **📚 CSS Selectors – The Styling Arsenal**  
- **Basic**:  
  - Element: `p` → All `<p>` tags.  
  - Class: `.nav` → `<div class="nav">`.  
  - ID: `#header` → `<div id="header">`.  
- **Advanced**:  
  - Descendant: `nav ul` → `<ul>` inside `<nav>`.  
  - Child: `div > p` → Direct `<p>` children of `<div>`.  
  - Attribute: `[data-type="primary"]` → Elements with that attribute.  
- **Pseudo**:  
  - `:hover` → Style on hover.  
  - `::after` → Add content after element.

🔹 **⚡ JavaScript Selectors – The Automation Strike**  
- **Basic**:  
  - `document.getElementById("header")` → Single ID.  
  - `document.getElementsByClassName("nav")` → All with class.  
- **Modern**:  
  - `document.querySelector(".nav")` → First match.  
  - `document.querySelectorAll("div > p")` → All matches.  
- **Example**:  
  ```javascript
  const biasData = document.querySelectorAll("[data-bias='flag']");
  biasData.forEach(item => console.log(item.textContent));
  ```

🔹 **🚀 Applications – Strike Examples**  
- **Scraping**: Use `querySelectorAll` to extract biased headlines—[[bias-flag-script]].  
- **Styling**: `nav ul li:hover` styles *Lion* UI for clarity.  
- **Accessibility**: Selectors ensure UI elements are accessible—e.g., ARIA labels—[[cybersecurity-activist-tools]].  
- **Justice**: Audit web design for bias—e.g., color contrast for inclusivity.

🔹 **⚙ Best Practices – Hunt with Precision**  
- **Specificity**: Favor `.nav-item` over generic `div`.  
- **Efficiency**: Use IDs for speed—`#header` over `.header`.  
- **Clarity**: Avoid over-nesting—`nav ul` not `div nav ul li`.  
- **Standards**: Align with *vault-standards.md*—clean, reusable selectors.
