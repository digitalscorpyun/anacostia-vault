## 🧠 **Core Python Concepts**

### **1️⃣ Syntax & Variables**

- **Indentation Rules**: Python uses spaces, not braces—clean and strict. Example: `if x > 0: print("Positive")`.  
- **Variables**: Dynamic typing (e.g., `x = 5`, `name = "Anacostia"`) kickstarts your scripts.  

🔗 **Related Notes**:  
- [[python-python-learning-path]] – Syntax drills.  
- [[python-overview]] – Broader Python scope.

### **2️⃣ Control Structures**

- **Conditionals**: `if`, `elif`, `else` steer logic—e.g., `if bias_score > 0.5: flag_bias()`.  
- **Loops**: `for` and `while` iterate data—e.g., `for article in news: analyze_sentiment(article)`.  

🔗 **Related Notes**:  
- [[oop-in-python]] – Loops in OOP.
