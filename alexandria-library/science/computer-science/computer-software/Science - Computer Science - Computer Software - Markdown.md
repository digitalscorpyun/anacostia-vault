#note_taking 
**Markdown** is a lightweight markup language used for formatting plain text. It’s designed to be easy to read and write, making it a popular choice for creating content that needs to be converted to HTML, such as documentation, blog posts, and README files.

---

### **Basic Markdown Syntax**

#### **Headings**

Use `#` for headings. Add more `#` symbols for smaller headings.

```markdown
# Heading 1
## Heading 2
### Heading 3
```

---

#### **Text Formatting**

- **Bold**: `**bold text**` → **bold text**
- _Italic_: `*italic text*` → _italic text_
- ~~Strikethrough~~: `~~strikethrough~~` → ~~strikethrough~~

---

#### **Lists**

- **Unordered List**:
    
    ```markdown
    - Item 1
    - Item 2
      - Subitem
    ```
    
- **Ordered List**:
    
    ```markdown
    1. First item
    2. Second item
    ```
    

---

#### **Links**

```markdown
[Text](https://example.com)
```

→ [Text](https://example.com/)

---

#### **Images**

```markdown
![Alt text](https://example.com/image.png)
```

---

#### **Code Blocks**

- Inline code: Use backticks: `` `code` ``
- Code block:
    
    def hello_world(): print("Hello, world!")
    

---

#### **Blockquotes**

```markdown
> This is a blockquote.
```

→

> This is a blockquote.

---

### **Why Use Markdown?**

- **Simplicity**: Minimal syntax, easy to write and read.
- **Portability**: Supported by tools like GitHub, Notion, and many CMS platforms.
- **Conversion**: Converts easily into HTML for web publishing.
