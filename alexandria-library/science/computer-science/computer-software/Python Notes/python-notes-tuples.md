In Python, **tuples** are one of the built-in data structures, similar to lists, but with a key difference: **tuples are immutable**, meaning their elements cannot be modified once the tuple is created. Python refers to values that cannot change as immutable, and an immutable list is called a tuple.
### **Key Features of Tuples**

1. **Immutable**: Once a tuple is created, its contents cannot be changed (no adding, removing, or changing elements).
2. **Ordered**: Tuples maintain the order of elements, making them indexable and iterable.
3. **Allow Duplicates**: Like lists, tuples can store duplicate values.
4. **Can Hold Multiple Data Types**: Tuples can contain any combination of data types, such as integers, strings, or even other tuples.
5. Tuples are created by enclosing elements in parentheses ()
- Creating tuples
	- empty_tuple = ()
	- single_element_tuple = (5,)  # Note the trailing comma for a single element
	- multiple_element_tuple = (1, "Hello", 3.14, [5, 6])

This shows the power of Python tuples—they can store elements of **different types** within the same structure, making them versatile for organizing complex data.
