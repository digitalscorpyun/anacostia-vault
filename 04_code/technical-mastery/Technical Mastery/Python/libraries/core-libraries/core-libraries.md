## 🧰 Core Python Libraries

|Library|Purpose|Usage Example|
|---|---|---|
|`os`|File/directory manipulation|Automate file movement in vault scripts|
|`sys`|Command-line arguments, interpreter|Handling script input/output|
|`datetime`|Date/time operations|Timestamp vault note creation|
|`re`|Regular expressions|Pattern matching for link validation|
|`json`|JSON parsing|Managing config files in Obsidian tools|
|`math`|Mathematical functions|Algebraic modeling in AI/ML notebooks|
|`logging`|Debug & audit logs|Track script executions|
