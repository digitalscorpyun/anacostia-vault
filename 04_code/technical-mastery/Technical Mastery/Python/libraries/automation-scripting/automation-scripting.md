# 🛠️ Python Automation Scripts

This note collects key scripts designed to automate maintenance, auditing, and vault generation tasks inside your Obsidian knowledge base.

---

## 📂 Script Categories

- **File Operations**: Read/write/delete files, navigate directories.
- **Scheduling Tasks**: Use `schedule`, `APScheduler`, or cron with Python.
- **Web Automation**: Automate browsing with Selenium or Requests + BeautifulSoup.
- **Obsidian Integration**: Create scripts for vault structure, note generation, link audits.
- **Cross-Platform Compatibility**: Write scripts that work on both Windows (PowerShell) and Linux (Bash).

---

## 🔍 Example Scripts

- `validate_backlinks.py` – Scans vault notes for broken links and orphaned files.
- `python-vault-reorganizer.ps1` – Reorganizes Python folders under `Technical Mastery`.
- `daily-note-automator.py` – Generates templated daily notes.
- `weekly-todo-populator.ps1` – Adds and logs weekly task cycles to `to-do-list.md`.
